# Music Genres and Mental Health

## Project Overview
This project explores the relationship between different music genres and mental health indicators such as mood, anxiety, and emotional wellbeing. 
The aim is to identify patterns and trends using data analysis and visualisation techniques.

## Research Question
How do different music genres affect mental health and emotional wellbeing?

## Approach
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualisation of genre vs mental health metrics
- Interpretation of observed trends

## Tools
- Power Query
- Power BI
- Excel
